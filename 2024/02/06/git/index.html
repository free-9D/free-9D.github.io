git命令

  • git init：初始化一个git仓库

  • git clone：复制一个git仓库

  • git config：配置信息

  • git add：添加文件到缓存命令

  • git status：查看文件的状态命令

  • git diff：查看更新的详细信息命令

  • git commit：提交命令

  • git reset HEAD：取消缓存命令

  • git rm：删除命令

  • git mv：移动或重命名命令
